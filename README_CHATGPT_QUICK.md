## 🔗 3 步连接

### 1. 打开 ChatGPT

访问: https://chat.openai.com

### 2. 创建连接器

1. **Settings** → **Connectors** → **Advanced**
2. 开启 **Developer mode**
3. 点击 **Create**
4. 填入上方信息
5. 点击 **Create**

### 3. 测试对话

新建对话 → **+** → **Developer mode** → 开启连接器

试试这个提示词：
```
请查询 2025 年 10 月 12 日的同工服侍安排
```
